# 磐石伺服器
磐石伺服器是一個架設在 Google Cloud Platform 的邀請制原味生存 Minecraft 伺服器。  
網路穩定快速，伺服器能力可以隨時根據需求調整。  

我們的目標是提供一個**長期**、**穩定**、**不延遲**的伺服器。  
目前由數位軟體工程師鎮守，務求提供優質的服務給各位。  

目前已經於 **1.17** 版本開放內部測試，歡迎加入我們。
